Projet web Map interactive

Pour ce projet, nous avons réalisé une application collaborative web avec une map interactive qui a pour thème les monuments dans le monde entier.

Nous avons choisi ce thème car il est important de nos jours de visiter des monuments qui nous relient au passé et nous permettent de nous cultiver. En visitant ces riches sources d’histoires où l’histoire s’est déroulée, nous trouvons nos racines. Notre site permet à l’utilisateur de se renseigner géographiquement sur ces derniers.
